# FR-HR-004 智能筛选 — 支持的条件参考

## 一、可筛选的字段

### 基础信息

| 字段 | 含义 | 示例描述 |
|------|------|----------|
| `name` | 姓名 | "姓王的候选人"、"名字包含'豪'" |
| `phone` | 手机号 | "手机号以138开头" |
| `id_card` | 身份证号 | (通常不用于筛选) |
| `gender` | 性别 (1=男, 2=女) | "男性" / "女性" |
| `age` | 年龄 | "30岁以下"、"25到40岁" |
| `education` | 学历 | "本科学历"、"大专及以上" |
| `address` | 居住地址 | "住在北京的" |
| `status` | 状态 | "面试中"、"已完成的候选人" |

### 驾驶相关

| 字段 | 含义 | 示例描述 |
|------|------|----------|
| `work_experience` | 工作年限/驾龄(年) | "驾龄3年以上"、"5年以上驾龄" |
| `qualification_detail.license_type` | 准驾车型 | "持有A2驾照"、"有A1驾照"、"B1或B2的" |
| `qualification_detail.license_years` | 从初次领证日期计算的驾龄 | "实际驾龄超5年" |
| `qualification_detail.qualification_valid` | 资格证是否在有效期内 | "资格证在有效期内"、"资格证过期" |

### 评分

| 字段 | 含义 | 示例描述 |
|------|------|----------|
| `total_score` | 总分 | "总分80分以上"、"得分最高的" |

---

## 二、支持的操作符

| 操作符 | 含义 | 适用场景 |
|--------|------|----------|
| `eq` | 等于 | 性别=男、学历=本科 |
| `ne` | 不等于 | 不是男性 |
| `gt` | 大于 | 驾龄 > 3年、总分 > 80 |
| `gte` | 大于等于 | 年龄 >= 25 |
| `lt` | 小于 | 年龄 < 35 |
| `lte` | 小于等于 | 总分 <= 60 |
| `contains` | 包含 | 姓名包含某个字、驾照包含A2 |
| `in_list` | 在列表中 | 状态在[面试中,已完成] |

---

## 三、推荐的筛选例句

| # | 输入 | 说明 |
|---|------|------|
| 1 | `驾龄3年以上、持有A2驾照且资格证在有效期内的男性候选人` | 组合条件：性别+驾龄+驾照+资格证 |
| 2 | `年龄30到40岁的A1驾照男性` | 年龄范围+驾照+性别 |
| 3 | `总分80分以上、已完成面试的候选人` | 评分类+状态类 |
| 4 | `本科学历、有B1或B2驾照的` | 学历+驾照 |
| 5 | `资格证在有效期内` | 纯资格证条件 |
| 6 | `驾龄不足2年的新司机` | 反向条件（lt） |
| 7 | `面试中或已完成、总分大于70` | OR逻辑+多种条件 |

---

## 四、逻辑关系

| 逻辑 | 含义 | 例句 |
|------|------|------|
| `AND` | 所有条件都必须满足 | "驾龄3年以上**且**持有A2驾照的男性" |
| `OR` | 满足任一条件即可 | "A1**或**A2驾照的候选人" |

---

## 五、状态枚举值

| 值 | 含义 |
|----|------|
| 0 | 待完善 |
| 1 | 材料已提交 |
| 2 | 面试中 |
| 3 | 已完成 |
| 4 | 已淘汰 |
